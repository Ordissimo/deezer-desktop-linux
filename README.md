# Requirement

* pepperflashplugin-nonfree
> [Debian](https://wiki.debian.org/PepperFlashPlayer/Installing)
> [Ubuntu](http://packages.ubuntu.com/fr/yakkety/pepperflashplugin-nonfree)

# How to run

```
npm install
npm start
```

# How to build

```
npm install
cd app && npm install && cd ..
npm run build
npm run dist
```
